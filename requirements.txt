# agentic-starter
Production-ready LangGraph + multi-agent workflow template (April 2026)

- Supervisor + Researcher + Critic + Executor
- Reflection / critique loop

## Quick Start
1. pip install -r requirements.txt
2. Copy .env.example → .env and add your OpenAI key
3. Run: python main.py
