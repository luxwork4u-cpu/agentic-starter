# agentic-starter
Production-ready LangGraph + multi-agent workflow template (April 2026)

**Features:**
- Supervisor + 3 specialized agents (Researcher, Critic, Executor)
- Built-in reflection / critique loop
- Persistent memory (thread-based)
- Structured Pydantic state
- Easy to extend with more agents & tools

## Quick Start
1. `pip install -r requirements.txt`
2. Copy `.env.example` → `.env` and add your OpenAI key
3. Run: `python main.py`

Built for solo developers who want a real agentic system that scales.
